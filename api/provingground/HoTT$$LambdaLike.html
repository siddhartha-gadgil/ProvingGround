LambdaLike

abstract class LambdaLike[X <: Term, +Y <: Term with Subs[Y]] extends FuncTerm[X, Y]

A lambda-expression. variable is mapped to value.

Refine to include a domain and codomain (maybe not - can have dependent type), and extend FuncObj.

Should have a trait with the basic properties and inductive definitions

Check: Is this the type? It may be a dependent type. The below is a lambda defined function.
